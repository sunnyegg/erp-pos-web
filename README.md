# ERP POS WEB

## Installation

```bash
$ git clone https://github.com/sunnyegg/erp-pos-web
$ cd [folder]
$ yarn add
```

## Environment Variables

```bash
$ nano .env
```

Example:

```
REACT_APP_API_URL = http://localhost:3333 // API URL
```

## Start

```bash
$ yarn start
```
